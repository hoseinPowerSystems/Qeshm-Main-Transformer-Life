# Qeshm-Main-Transformer-Life
# Remaining Life Estimation of Qeshm Power Plant Main Transformers

This project estimates the **remaining life** of the main transformers of gas units in the **Qeshm Combined Cycle Power Plant**.

## Objective
The main goal is to assess transformer lifetime for:
- Maintenance planning
- Reliability improvement
- Asset management
- Replacement decision support

## Methodology
The proposed approach is based on:
- **Monte Carlo simulation**
- **Combined lifetime modeling**
- **Thermal stress evaluation**
- **Electrical stress evaluation**
- Aging and failure-rate analysis

## Key Features
- Plant-specific transformer life estimation
- Probabilistic uncertainty modeling
- Reliability-oriented analysis
- Support for maintenance and replacement planning

## Files
- `remaining_life_estimation.m` — MATLAB implementation
- `results.png` — Remaining life estimation results
- `methodology_flowchart.png` — Workflow of the proposed method
- `paper_reference.pdf` — Reference report/article

## Results
The model outputs:
- Estimated remaining life of each transformer
- Probabilistic lifetime assessment
- Decision support for maintenance prioritization

## Applications
- Power plant transformer asset management
- Condition-based maintenance
- Reliability assessment in industrial power systems
